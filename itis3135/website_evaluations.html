Web Design Evaluation - Khamryn Turner

Website Evaluations

Apple

Apple's website is a great example of modern web design with a sleek and user-friendly layout.

  • Contrast: Excellent use of color contrast, making key elements stand out.
  • Repetition: Consistent use of fonts, colors, and design elements throughout.
  • Alignment: Well-structured grid layout ensuring a clean and professional look.
  • Proximity: Related elements are grouped logically, improving readability.

Ling's Cars

This site is known for its chaotic and overwhelming design, making it a poor example of good web design principles.

  • Contrast: Overuse of bright, clashing colors makes reading difficult.
  • Repetition: Inconsistent fonts and styles create a disorganized feel.
  • Alignment: Random placement of elements leads to visual clutter.
  • Proximity: Unrelated items are placed too close together, making navigation confusing.
